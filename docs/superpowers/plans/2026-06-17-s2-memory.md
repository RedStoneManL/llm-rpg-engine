# S2: Memory — LLM provider + importance + recall ranking + reflection — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development / executing-plans. `- [ ]` steps, TDD.

**Goal:** The memory subsystem: a provider-agnostic `LLMProvider` switchboard (codingplan/Anthropic + 质谱/Zhipu + OpenAI, + `FakeLLMProvider` for tests), an importance scorer (heuristic floor + LLM rubric), recall ranking (recency × importance × relevance, reusing the local `engine.embed` model), and reflection (importance-accumulation trigger → LLM synthesis → write-back facts).

**Architecture:** Backstage cognition runs through `LLMProvider` (never the main loop directly). Offline tests use `FakeLLMProvider` (deterministic, no network) — real adapters are thin stdlib-`urllib` HTTP wrappers, exercised only for request-formatting offline (live calls need keys → integration, deferred). Embeddings reuse `engine.embed.get_embedder()` (FakeEmbedder offline, fastembed bge-small-zh for real). Importance/recall/reflection are pure where possible; the LLM-dependent bits take an injected `LLMProvider` so they're fully fakeable.

**Tech Stack:** Python 3.12 stdlib (+ `engine.embed`/`engine.vectorstore`/`engine.archive` reuse). Offline tests `python3 -m pytest -q --ignore=tests/test_embed_real.py`. Pure stdlib; do NOT install SDKs (adapters use `urllib.request`).

**Conventions/Guardrails:** TDD per task; `get_logger`; commit per task; `python3`. Never `git init`/`rm -rf .git`/`checkout --orphan`; never delete/modify `_legacy/` or `docs/`; do NOT touch `kernel/`. New packages `llm/`, `memory/`.

## Task 3: Importance scorer
`memory/importance.py`: `heuristic_floor(event) -> int` (0–10: +floor if event touches a tracked entity / opens a thread / changes a tracked fact — inspect event type+deltas); `score(event, *, provider=None) -> int`: max(heuristic_floor, LLM rubric score if provider given). LLM rubric prompt: anchored 1–10 (1 寒暄/赶路 · 3 有信息对话 · 6 关系转折 · 8 重大抉择 · 10 背叛/死亡); parse int, clamp 1–10. With `provider=None` → heuristic only.
- [ ] Step1 failing tests (use FakeLLMProvider): heuristic_floor gives a higher floor to a thread-opening event than a trivial one; `score(event, provider=fake_returning_"8")` → 8; `score(trivial, provider=None)` → heuristic value; clamps out-of-range. Step2 fail. Step3 implement. Step4 pass. Step5 commit `feat(memory): importance scorer (heuristic floor + LLM rubric)`.

## Task 4: Recall ranking (recency × importance × relevance)
`memory/recall.py`: `rank(candidates, query_vec, *, now_day, embedder=None) -> list[(candidate, score)]` where each candidate has `{text, day, importance, vec?}`. score = `w_rec*recency(now_day,day) * + w_imp*norm(importance) + w_rel*cos(query_vec, cand_vec)` (use a weighted sum; recency = exp-decay or 1/(1+age)). Relevance via cosine (reuse `engine.vectorstore` cosine or numpy). Provide `embed_query(text, embedder)` reusing `engine.embed.get_embedder()` (FakeEmbedder offline). Pure ranking given components.
- [ ] Step1 failing tests (FakeEmbedder): a recent high-importance on-topic candidate ranks above an old low-importance off-topic one; weights make each axis matter (toggle one axis, ordering changes). Step2 fail. Step3 implement (numpy cosine; document the weights as module constants). Step4 pass + full suite green. Step5 commit `feat(memory): recall ranking recency×importance×relevance`.

## Task 5: Reflection (trigger + synthesis)
`memory/reflection.py`: `should_reflect(accumulated_importance, *, threshold=30) -> bool` (importance-accumulation trigger); `reflect(subject, recent_events, *, provider) -> dict` → LLM synthesizes a higher-level shift, returns `{"predicate":"arc","value":"<summary>"}` (a fact to assert on the subject via the character/ontology write path — reflection PRODUCES a fact-delta, the caller asserts it). Skip subjects flagged 纯粹/recently-reflected (caller's concern; `reflect` just synthesizes).
- [ ] Step1 failing tests (FakeLLMProvider): should_reflect crosses at threshold; `reflect("艾拉", events, provider=fake)` returns an `arc` fact-delta dict using the fake's synthesis. Step2 fail. Step3 implement. Step4 pass + full suite green. Step5 commit `feat(memory): reflection trigger + LLM arc synthesis`.
